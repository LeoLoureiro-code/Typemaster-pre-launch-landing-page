# Typemaster Landing Page

## 📌 Overview
This project is a responsive landing page built to practice modern CSS layout techniques and responsive design.

## 🚀 Features
- Fully responsive layout
- Mobile-first design
- Optimized images using `<picture>`
- Clean and semantic HTML
- SEO 90+

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
